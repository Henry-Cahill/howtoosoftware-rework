# Images Directory

## Logo File

Place your white HTS logo here as: **hts-logo-white.png**

The logo should be:
- PNG format with transparent background
- White or light colored (shows well on purple gradient)
- Approximately 50x50px or higher resolution
- Square or similar aspect ratio

The theme will automatically load this logo in:
- Header (top left)
- Footer (brand section)

## Current Setup

The theme is configured to use: `{{asset 'images/hts-logo-white.png'}}`

This will resolve to: `/assets/images/hts-logo-white.png`
